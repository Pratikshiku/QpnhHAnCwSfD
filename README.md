### 三、功能描述
系统分为（身份）：普通用户（学生）、教师、企业、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 就业信息管理
6. 实习信息管理
7. 老师管理、老师发布的公告管理
8. 企业管理、企业发布的公告管理

**教师功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 就业信息管理
5. 实习信息管理
6. 老师发布的公告管理
7. 企业管理、企业发布的公告管理
8. 学生管理


**企业功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 就业信息管理
5. 实习信息管理
6. 老师发布的公告管理
7. 企业发布的公告管理




**管理员功能：**
1. 登录
2. 人中心（修改密码、个人信息）
3. 基础数据管理（班级管理、老师公告类型管理、企业公告类型管理、行业管理、实习信息类型管理、院系管理）
4. 就业信息管理
5. 实习信息管理
6. 老师管理
7. 企业管理
8. 学生管理
